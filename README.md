# Jogo da Forca (Python)
Um simples jogo da forca desenvolvido em **Python**, jogável diretamente no terminal.  
Este projeto foi criado com o objetivo de praticar lógica de programação, controle de fluxo e manipulação de strings.

---

## Sobre o Projeto
O jogador deve adivinhar a palavra secreta, letra por letra.  
A cada erro, o bonequinho da forca vai sendo desenhado até o limite de 6 tentativas.

Funções principais:
- Escolha aleatória de palavras com `random.choice()`
- Exibição visual da forca via ASCII (`display_hangman()`)
- Controle de tentativas e letras erradas
- Uso de `list comprehension` e estruturas de repetição
